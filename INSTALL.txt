Installation instructions to see the current progress:

Advanced users: Make sure you've got python 3.2.* (32-bit) and pygame installed, clone the repo and run main.py.

Detailed instructions (Windows):
Install python 3.2.5 from https://www.python.org/ftp/python/3.2.5/python-3.2.5.msi
Install pygame 1.9.2a0 for PY3.2 from http://pygame.org/ftp/pygame-1.9.2a0.win32-py3.2.msi

Go to the project homepage: https://github.com/ZeeQyu/TileGame
and press "Download Zip" Save the file to somewhere and extract the entire folder.
Run main.py.

That should do it. Mind the stairs.
